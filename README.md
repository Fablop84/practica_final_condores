# Chile Compra - Transformando las licitaciones de mercado en patrones predecibles

Este proyecto corresponde al **Proyecto Final** del Bootcamp Big Data, Inteligencia Artificial & Machine Learning | Edición XVI.  
El objetivo es desarrollar un modelo de **predictibilidad para el mercado de licitaciones del estado de Chile** capaz de clasificar diferentes categorías de licitaciones, optimizando la precisión mediante técnicas avanzadas de entrenamiento.

# Chile Compra Project - Dataset

Este proyecto utiliza un enfoque de Machine Learning Clásico para procesar:
- **Datos**: Descarga de información de la API de mercado libre de Chile entre los años 2020 - 2025.
